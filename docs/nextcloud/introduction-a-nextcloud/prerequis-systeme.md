* ✅ : Compatible et recommandé
* ⚠️ : Compatible avec limitations
* ❌ : Non compatible

| ---------------------- | ----- | ------- |
| Ubuntu                 | 22.04 | ✅       |
|                        | 24.04 | ✅       |
| Debian                 | 12    | ✅       |
|                        | 13    | ✅       |
